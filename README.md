

This repo introduces factors that influence Canadians' stress level based on "General social survey on Canadians at Work and Home (cycle 30), 2016 ¡± survey published by Canadian General Social Survey (GSS). 
The use of raw data is restricted due to confidentiality reasons. However, if you want to download the data and do your own analysis, you can access the data by submitting a request form on  Canadian GSS.


## File Structure

Inputs contains the raw data from Canadian GSS.

Outputs are consist of the rmd and pdf file of our paper "Factors Influence Stress in Canada".

Scripts are consist of the R script about data cleaning.
